> Gym is a web application that was created for personal use. In it you can create your workout routines, add your exercises and track your development.


## Prerequisites

- php >=7.2.0
- composer >= 1.9.0
- npm >= 6.0.0

## Install

```sh
cp .env.example .env
composer install
npm install && npm run dev
php artisan key:generate
php artisan migrate
```

## Usage

```sh
php artisan serve
```
